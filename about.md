# Keybinds

There's keybinds for no reason but it's cool.

* **Backspace** — Remove point
* **Enter** — Add point
* **1** — Toggle main color
* **2** — Toggle secondary color
* **3** — Toggle glow color
* **4** — Toggle line color
* **5** — Toggle white color
* **L** — Toggle lock gradient
* **C** — Copy
* **V** — Paste
* **S** — Save
* **O** — Load
* **Arrow Up** — Move up
* **Arrow Down** — Move down
* **Arrow Left** — Move left
* **Arrow Right** — Move right
